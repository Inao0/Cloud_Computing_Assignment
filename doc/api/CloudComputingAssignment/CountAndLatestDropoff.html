CountAndLatestDropoff 

case class CountAndLatestDropoff(count: Int, latest_dropoff: Timestamp) extends Ordered[CountAndLatestDropoff] with Product with Serializable

Contains a count and a latest drop-off timestamp Meant to be corresponding to a timeWindow and Route

Implements the ordered traits to be able to use the max function of scala to find the most frequented route as defined by the debs 2015 challenge in the query 1 : the most of frequent route over a time window is the one with the highest number of drop-off during the ime window. In case of equality in the number of drop-off the second criterion is the most recent drop-off goes first.

count

: number of drop-off (for a given route and time-window)

latest_dropoff

: most recent drop-off (for a given route and time-window)
